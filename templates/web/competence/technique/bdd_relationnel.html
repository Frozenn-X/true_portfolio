Les bases de données relationnelles

Les bases de données relationnelles sont des bases de données où l’information est organisée dans des tableaux à deux dimensions qu’on appelle des tables. Les tables contiennent toutes les relations entre les différentes données telles qu’un type de produit relié à une table de produit. Les bases de données relationnelles permettent simplement et efficacement d’obtenir une multitude de données stockées concernant une donnée, des “clefs étrangères” permettent la relation entre plusieurs tables. Le langage SQL (Structured Query Language) est un exemple de langage normalisé permettant d’exploiter ces bases de données relationnelles. La capacité d’interrogation est une force de ces bases de données, elle permet d’accomplir des recherches complexes de manière intuitive et simplifiée.

Dans le cadre de ma formation, j’ai utilisé à de nombreuses reprises MariaDB. MariaDB est une des solutions de base de données relationnelles open source créée par les développeurs originaux de MySQL (MySQL étant un système de gestion de base de données relationnelles). Nous utilisons le logiciel MySQL Workbench pour pouvoir effectuer nos opérations via une interface graphique. Après avoir mis en place le schéma de données que nous allions utiliser, ainsi que le service MariaDB sur notre serveur Linux, j’avais comme objectif d’exploiter des données d’un échantillon de personnes, donnant suite à une analyse sous forme d’un reporting Power BI. La mise en place de schéma de données peut être très complexe dans certains cas et peut avoir un impact très conséquent sur le bon fonctionnement d’une application par exemple. Ce qui me fait penser à l’anecdote suivante, nous avons un projet client qui travaille pour un concessionnaire automobile dans le marketing, qui voulait implémenter un système d’historisation de données (concernant les concessions, un concessionnaire automobile en France), en utilisant mes compétences en base de données relationnelles j’ai pu adapter le schéma de données pour rajouter cette fonctionnalité via une interface sous la forme d’un formulaire en ligne.
